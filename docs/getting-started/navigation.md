## Header Bar

The header bar displays:

- **Page Title** - Current page name and description
- **Notifications** - Bell icon showing unread notifications
- **Help** - Question mark icon for contextual help
- **Page Actions** - Context-specific buttons (e.g., "New Workflow", "Export")

### Notifications

Click the bell icon to see:
- Session updates
- Observation notifications
- Invitation alerts
- System messages

**Clear Notifications:**
1. Click the bell icon
2. Click **"Mark all as read"** to clear the badge
3. Individual notifications can be dismissed by clicking them

---

## Collapsible Sidebar

On desktop:
- Click the **chevron** (< or >) to collapse/expand the sidebar
- Collapsed mode shows only icons with tooltips

On mobile:
- Use the **hamburger menu** (â˜°) to open the sidebar
- Sidebar appears as a slide-out drawer
- Tap outside to close

## Breadcrumb Navigation

On detail pages, you'll see breadcrumbs showing your location:

```
Workflows > Patient Intake Process > Edit
```

Click any breadcrumb segment to navigate back.

---

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `?` | Open help panel |
| `Esc` | Close dialogs/modals |

---

## Page-Specific Navigation

### Workflow Editor
- **Zoom Controls** - Bottom right corner
- **Minimap** - Overview of large workflows
- **Node Palette** - Left panel for adding steps

### Session View
- **Step Navigator** - Click steps in the workflow to view/add observations
- **Observation Panel** - Right side panel for tagging waste

### Analytics
- **Session Selector** - Dropdown to filter by specific session
- **Tab Navigation** - Switch between chart types

---

## Mobile Navigation

On mobile devices:
- Sidebar becomes a slide-out drawer
- Tables become scrollable cards
- Actions move to dropdown menus
- Touch-friendly buttons and controls

---

## Tips for Efficient Navigation

1. **Use the sidebar** for main section navigation
2. **Use breadcrumbs** to go back in multi-level pages
3. **Collapse sidebar** on smaller screens for more content space
4. **Bookmark the cheat sheet** for quick access during sessions
5. **Check notifications** regularly for session updates
