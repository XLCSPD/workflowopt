## Quick Overview

ProcessOpt is a comprehensive Lean process improvement platform designed to help teams identify and eliminate waste in their workflows. The platform supports:

- **Interactive Training** - Learn the 8 types of Lean waste (DOWNTIME) plus digital waste categories
- **Visual Workflows** - Map your processes with an intuitive drag-and-drop editor
- **Collaborative Sessions** - Run waste walks with your team in real-time
- **Actionable Analytics** - Get AI-powered insights and prioritized improvement recommendations

## User Roles

| Role | Description |
|------|-------------|
| **Participant** | Join sessions, tag waste observations, complete training |
| **Facilitator** | Create/manage sessions and workflows, view analytics |
| **Admin** | Full access including user management and system settings |

## Getting Help

- Click the **Help** button (?) in the top navigation bar for context-aware assistance
- Use the **Waste Cheat Sheet** in the sidebar for quick waste type references
- Contact your organization administrator for account issues
