# Training Modules Guide

Learn how to use the training system to master Lean waste identification.

## Overview

The Training Library provides structured learning content to help you:
- Understand Lean methodology principles
- Identify the 8 types of waste (DOWNTIME)
- Recognize digital-specific waste categories
- Apply waste identification in real processes

2. Complete the content (watch video, read article, take quiz)

## Waste Categories

### Core Lean Wastes (DOWNTIME)

| Code | Name | Description |
|------|------|-------------|
| **D** | Defects | Errors requiring rework or correction |
| **O** | Overproduction | Making more than needed |
| **W** | Waiting | Idle time, delays |
| **N** | Non-utilized talent | Underused skills and knowledge |
| **T** | Transportation | Unnecessary movement of materials |
| **I** | Inventory | Excess stock or work-in-progress |
| **M** | Motion | Unnecessary movement of people |
| **E** | Extra processing | Doing more than required |

### Digital Waste Categories

| Code | Name | Description |
|------|------|-------------|
| **IW** | Information Waste | Poor data quality, missing information |
| **DO** | Digital Overload | Too many tools, notifications, systems |
| **UF** | Unused Features | Software capabilities not utilized |
| **ED** | Excessive Data | Collecting more data than needed |
| **FW** | Format Waste | Converting between incompatible formats |
| **DW** | Digital Waiting | System lag, slow processing |

## Tips for Success

1. **Complete training before sessions** - You'll be more effective at identifying waste
2. **Take notes** - Write down examples relevant to your work
3. **Practice identifying waste** - Look for examples in your daily work
4. **Use the cheat sheet** - Reference it during sessions
5. **Retake quizzes** - Review modules periodically to refresh knowledge
