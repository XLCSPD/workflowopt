# Organization Settings

Learn how to configure and manage your organization settings as an administrator.

## Overview

Organization settings allow you to:
- Update organization information
- Configure system preferences
- View organization statistics
- Manage data and integrations

---

## Accessing Organization Settings

1. Click **Admin** in the sidebar
2. Select the **Organization** tab
3. View and edit organization settings

---

## Organization Profile

### Organization Name
The name displayed throughout the application.

**To Update:**
1. Edit the **Name** field
2. Click **"Save Changes"**

Your organization name appears in:
- Page headers
- Reports and exports
- User invitation emails

### Organization Description
Optional text describing your organization or team.

### Logo
Upload your organization logo (if supported):
- Recommended size: 200x200 pixels
- Formats: PNG, JPG, SVG
- Used in reports and branding

---

## Statistics Overview

View key metrics for your organization:

### User Statistics
- **Total Users** - Active members count
- **By Role** - Breakdown of admins, facilitators, participants
- **Pending Invitations** - Unaccepted invites

### Content Statistics
- **Workflows** - Total workflows created
- **Sessions** - Completed sessions count
- **Observations** - Total observations recorded

### Training Statistics
- **Completion Rate** - Overall training progress
- **Active Learners** - Users currently in training
- **Modules Available** - Training content count

---

## Configuration Options

### Default Settings

**Default User Role**
The role assigned to new users who self-register (if enabled).
- Options: Participant, Facilitator
- Recommendation: Participant (safest default)

**Session Settings**
- Allow participants to end sessions
- Require minimum observations
- Auto-archive after X days

### Notification Defaults
Set organization-wide notification defaults:
- Session notifications enabled
- Observation notifications enabled
- Weekly summary emails

Individual users can override these in their settings.

---

## Data Management

### Data Export

Export organization data for backup or analysis:

1. Click **"Export Data"**
2. Select what to include:
   - Users list
   - Workflows
   - Sessions and observations
   - Training progress
3. Choose format (CSV, JSON)
4. Download the export

### Data Retention

Configure how long data is retained:
- **Active data** - Retained indefinitely
- **Archived sessions** - Retained for X months
- **Deleted content** - Purged after 30 days

### Data Deletion

Request complete data deletion:
1. Contact system administrator
2. Confirm deletion request
3. All organization data removed
4. This action is irreversible

## Troubleshooting

### "Cannot save organization settings"
- Check you have admin permissions
- Verify required fields are filled
- Check for validation errors

### "Statistics not loading"
- Refresh the page
- Check for connectivity issues
- Contact support if persists

### "Export failed"
- Try smaller data range
- Check available storage
- Try different format

### "Logo not displaying"
- Verify file format
- Check file size limits
- Try re-uploading
