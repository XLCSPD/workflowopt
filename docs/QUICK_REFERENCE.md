## The 8 Wastes (TIMWOODS)

| Code | Waste Type | Key Indicator | Example |
|------|------------|---------------|---------|
| **T** | Transportation | Unnecessary movement of materials/info | Email chains, file transfers |
| **I** | Inventory | Excess work-in-progress | Backlog, queued requests |
| **M** | Motion | Unnecessary people movement | Walking to printer, searching files |
| **W** | Waiting | Idle time, delays | Approval bottlenecks, system loading |
| **O** | Overproduction | More than needed | Unused reports, extra copies |
| **O** | Overprocessing | More work than required | Excessive approvals, over-documentation |
| **D** | Defects | Errors requiring rework | Data entry mistakes, returns |
| **S** | Skills | Underutilized talent | Manual tasks that could be automated |
